### Ejercicio
#### Objetivo:

Realizar un programa que solicite cinco números e imprima por pantalla el promedio, el máximo y el mínimo.

- Version: 0.1 del 29 diciembre de 2015
- Autor: Mauricio Dávila
- Revisión: Ernesto Gigliotti

#### Aspectos a destacar:
*   El uso de bucles para pedir un número tras otro
*   El uso de acumuladores, se tiene que guardar la suma para luego hacer el promedio
*   El uso del if para preguntar si el numero ingresado es mayor o menor para guardar los máximos y mínimos
