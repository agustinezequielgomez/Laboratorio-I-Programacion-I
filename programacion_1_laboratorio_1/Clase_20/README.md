### Ejercicio 1
#### Objetivo:

    Ingresar datos de 1 persona e imprimirlos. Crear la estructura "Persona":
        nombre [32]
        edad
    Si se ingresa “salir” como nombre, se debe salir del programa, de lo contrario se pediran los datos de nuevo.
